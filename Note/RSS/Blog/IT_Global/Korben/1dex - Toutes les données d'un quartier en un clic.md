<p>Quentin, fidèle lecteur de Korben, développe en solo depuis presque un an un outil qui va parler <strong>à tous ceux qui cherchent un appart ou une maison</strong> et qui en ont marre de jongler entre quinze onglets pour avoir une vision claire d'un quartier.</p>
<p>
<a href="https://1dex.fr">1dex.fr</a>
c'est une plateforme qui agrège un paquet de données géographiques et immobilières sur n'importe quelle adresse en France. Prix de vente au m², transactions DVF, permis de construire, qualité de l'eau, pollution de l'air, travaux à proximité, écoles... Le tout sur une interface cartographique plutôt bien foutue.</p>
<img src="https://korben.info/cdn-cgi/image/width=1200,fit=scale-down,quality=90,f=avif/1dex-donnees-immobilieres-geographiques/1dex-donnees-immobilieres-geographiques-1.jpeg" alt="" loading="lazy" decoding="async">
<p>Concrètement, vous entrez une adresse, vous cliquez sur &quot;<em>Analyser cette zone</em>&quot; et hop, la carte se remplit de données. On peut alors visualiser les parcelles alentours, voir les dernières ventes, repérer les chantiers en cours, et même afficher les immeubles avec syndic de copropriété. Y'a même un système de calques pour switcher entre fond de carte classique, vue satellite ou mode sombre.</p>
<img src="https://korben.info/cdn-cgi/image/width=1200,fit=scale-down,quality=90,f=avif/1dex-donnees-immobilieres-geographiques/1dex-donnees-immobilieres-geographiques-1.png" alt="" loading="lazy" decoding="async">
<img src="https://korben.info/cdn-cgi/image/width=1200,fit=scale-down,quality=90,f=avif/1dex-donnees-immobilieres-geographiques/1dex-donnees-immobilieres-geographiques-2.png" alt="" loading="lazy" decoding="async">
<p>Perso j'aime bien ce genre d'outil qui met la data à portée de main sans avoir besoin de fouiller data.gouv.fr pendant des heures.</p>
<p>Son modèle est freemium mais rassurez-vous, l'essentiel est gratuit avec une limite journalière d'analyses. Si vous dépassez, soit vous revenez le lendemain, soit vous passez à la caisse pour un accès intensif. Quentin bosse aussi sur une API pour les pros et une extension navigateur qui ajoutera les données 1dex directement sur les annonces immo. Pas mal pour éviter les mauvaises surprises avant même de visiter !</p>
