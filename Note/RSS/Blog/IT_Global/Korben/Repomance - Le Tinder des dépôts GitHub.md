<p>Si comme moi, vous êtes du genre à passer des heures sur GitHub pour dénicher la perle rare, genre un nouveau framework à la mode ou l'outil CLI qui va vous sauver la vie, j'ai trouvé un truc qui va soit booster votre productivité, soit ruiner votre temps libre.</p>
<p>Ça s'appelle <strong>
<a href="https://apps.apple.com/us/app/repomance/id6756920720">Repomance</a>
</strong> et c'est une application iOS qui transforme la découverte de dépôts GitHub en une expérience... disons, plus familière.</p>
<img src="https://korben.info/cdn-cgi/image/width=1200,fit=scale-down,quality=90,f=avif/repomance-github-swipe-ios/repomance-github-swipe-ios-2.png" alt="" loading="lazy" decoding="async">
<p>J'vais pas y aller par 4 chemins, en fait c'est le Tinder du code. Vous avez une pile de cartes représentant des projets open source, et vous swipez. À droite, vous mettez une &quot;star&quot; au dépôt (ça se synchronise direct avec votre compte GitHub via OAuth), et à gauche, vous passez votre chemin.</p>
<p>L'appli vous propose deux modes : les projets &quot;<strong>Trending</strong>&quot; pour voir ce qui excite la communauté en ce moment (par jour, semaine ou mois), et un mode &quot;<strong>Curated</strong>&quot; qui vous prépare des petits lots personnalisés selon vos langages de prédilection et vos centres d'intérêt (Web, Machine Learning, DevOps, etc.).</p>
<img src="https://korben.info/cdn-cgi/image/width=1200,fit=scale-down,quality=90,f=avif/repomance-github-swipe-ios/repomance-github-swipe-ios-3.png" alt="" loading="lazy" decoding="async">
<p>C'est assez génial pour faire sa veille technique dans le métro ou entre deux réunions. Les fiches sont également super complètes avec description, stats, répartition des langages, preview du README et même la licence du projet.</p>
<p>Comme ça, pas besoin de sortir de l'appli pour savoir si le projet est maintenu ou s'il correspond à vos besoins. Et pour ceux qui se posent la question de la vie privée,
<a href="https://github.com/mpospirit-apps/Repomance-iOS">l'appli est open source</a>
et ne demande que le strict minimum de permissions GitHub pour pouvoir ajouter les étoiles à votre profil.</p>
<p>Voilà, c'est gratuit, c'est propre, et c'est totalement addictif quand on aime explorer la jungle de l'open source !</p>
